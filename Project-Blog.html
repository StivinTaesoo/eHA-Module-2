Web Development Fundamentals

My web development journey began with an introductory course on udemy which gave me an overview knowledge of web fundamentals such as website development, internet protocol (IP) address, hypertext transfer protocol (HTTP), coding, front-end/backend, content managament system, HTML,CSS and Javascript.

The knowledge gained from this course laid the foundation for what has become my remarkable journey in web development.

UI/UX Design

This course on the eHA Academy learning platform broadened my insight on design as I got to learn key design prinicipes such balance, contrast, emphasis and pattern.

I also got acquainted with concepts like human-centered design which entails empathy, user research and testing.
Vital knowledge on design thinking such as problem-solving, ideation and prototyping have become key components of my web development journey thus far.

HTML

I've come to learn that html is the building block for any website we see on the internet. It's been interesting to getting to understanding html structure - the doctype declaration, html, head and body tags. I have learnt essential html tags such as headings, paragraphs, links, list and others. It's quite exciting to put this tags together and come up with a meaningful website.

CSS

While html is the building block for websites, CSS is what gives the website the requisite appeal which is achieved from a good use of color, font-sizing layouts, font-styles and the likes.

I have come to learn and understand basic CSS syntax such as selectors, properties and values. understanding CSS units and selectors has been a key cog in my journey so far.
